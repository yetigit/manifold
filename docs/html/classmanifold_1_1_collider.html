template SparseIndices Collisions< Box > ( const VecDH< T > &  queriesIn) const

For a vector of query objects, this returns a sparse array of overlaps between the queries and the bounding boxes of the collider. Queries are normally axis-aligned bounding boxes. Points can also be used, and this case overlaps are defined as lying in the XY projection of the bounding box.
